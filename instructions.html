Setup Instructions for the Google API Java Client Library

Browse Online

  • Browse Source

Checkout Instructions

Prerequisite: install Java 5. You may need to set your JAVA_HOME.

Command-Line

Prerequisite: install Mercurial and Maven.

hg clone https://google-api-java-client.googlecode.com/hg/ google-api-client
cd google-api-client
mvn compile

Eclipse 3.5

Prerequisite: install Eclipse, the Mercurial plugin, the Maven plugin, and the Android plugin.

  • Preferences: Android; setup SDK location
  • SVN checkout
    • File > Import...
    • "Select"
      • Select "Mercurial > Clone Existing Mercurial Repository"
      • Click on "Next"
    • "Clone repository"
      • URL: https://google-api-java-client.googlecode.com/hg/
      • Clone directory name: google-api-client
      • Click on "Next"
    • "Select working directory revision"
      • Click on "Next"
    • "Import Projects"
      • Click on "Finish"
  • Install
    • Right-click on project google-api-client
    • Run As > Maven package

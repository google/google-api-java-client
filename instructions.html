Checkout Instructions

Prerequisite: install Java 5. You may need to set your JAVA_HOME.

Prerequisite: install Mercurial and Maven.

Command-Line

hg clone -r 1.2.1-alpha https://google-api-java-client.googlecode.com/hg/ google-api-java-client
cd google-api-java-client
mvn compile

Eclipse 3.5

Prerequisite: install Eclipse, the Mercurial plugin, the Maven plugin, and the Android plugin.

  • Setup Eclipse Preferences
    • Window > Preferences... (or on Mac, Eclipse > Preferences...)
    • Select Maven
      • check on "Download Artifact Sources"
      • check on "Download Artifact JavaDoc"
    • Select Android
      • setup SDK location
  • Checkout Hg project
    • File > Import...
    • "Select"
      • Select "Mercurial > Clone Existing Mercurial Repository"
      • Click on "Next"
    • "Clone repository"
      • URL: https://google-api-java-client.googlecode.com/hg/
      • Clone directory name: google-api-client
      • Click on "Next"
    • "Select working directory revision"
      • 1.2.1-alpha
      • Click on "Next"
    • "Import Projects"
      • Click on "Finish"
  • Import google-api-client project
    • File > Import...
    • Select "General > Existing Project into Workspace" and click "Next"
    • Click "Browse" next to "Select root directory", find google-api-client under existing google-api-java-client project directory and click "Next"
    • Click "Finish"
  • Compile
    • Right-click on project google-api-client
    • Run As > Maven build

Generator Instructions using Eclipse

For these instructions, you will need to first install Eclipse and Subversion . You will also need to ask to be added as a Committer (Google engineers only).
  • svn checkout https://gdata-java-client.googlecode.com/svn/branches/2 /tmp/gdata-java-client
  • start Eclipse
  • Workspace location: /tmp/gdata-java-client
  • Import /tmp/gdata-java-client/gdata
    • File > Import...
    • General > Existing Projects into Workspace
    • Next
    • Select root directory: /tmp/gdata-java-client/gdata
    • press <tab> and Finish
  • Import /tmp/gdata-java-client/gdata-generator
  • Run > Run Configurations
  • Select Java Application/Generate
  • Run (it should say "All files up to date.")
  • Edit the files in the directory /gdata-generator/data
  • Run > Run History > Generate
  • right-click on gdata project, Refresh
  • cd /tmp/gdata-java-client
  • svn add any file you may have added
  • svn commit -m "some informative comment"

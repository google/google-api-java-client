Google's legacy ClientLogin authentication method as described in ClientLogin for Installed Applications .

This package depends on theses packages:

  • {@link com.google.api.client.googleapis}
  • {@link com.google.api.client.googleapis.auth}
  • {@link com.google.api.client.http}
  • {@link com.google.api.client.util}

Warning: this package is experimental, and its content may be changed in incompatible ways or possibly entirely removed in a future version of the library

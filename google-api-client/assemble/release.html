Release Process

  • Update instructions.html
  • Assemble for Distribution: mvn clean javadoc:javadoc assembly:assembly
  • Upload to http://code.google.com/p/gdata-java-client/downloads/entry
    • Summary: Google API Java Client Library 1.0.0-alpha
    • File: gdata-java-client/gdata/target/google-api-client-1.0.0-alpha-java.zip
    • Labels: Type-Archive, OpSys-All, and Featured
    • click Submit file
  • Deprecate any obsolete libraries.
  • Create SVN Tag 
    svn cp -m "Tagging 1.0.0-alpha" https://gdata-java-client.googlecode.com/svn/branches/2 https://gdata-java-client.googlecode.com/svn/tags/google-api-client/1.0.0-alpha
  • Upload JavaDoc into Tag 
    svn checkout https://gdata-java-client.googlecode.com/svn/tags/google-api-client/1.0.0-alpha /tmp/tag
cd gdata-java-client/gdata
cp -R target/site/apidocs /tmp/tag/javadoc
cd /tmp/tag
svn add javadoc
cd javadoc
find . -name "*.html" -type f -print0 | xargs -0 svn propset svn:mime-type text/html
find . -name "*.css" -type f -print0 | xargs -0 svn propset svn:mime-type text/css
cd ..
svn commit -m "JavaDoc for google-api-client-1.0.0-alpha"
cd ..
rm -rf tag
  • Put it up on Maven central repository
  • Update http://code.google.com/p/gdata-java-client/admin

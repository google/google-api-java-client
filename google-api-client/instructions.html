Installation Instructions

Prerequisite: you must first install Java 5. You may need to set your JAVA_HOME.

If you need commit access, you first need to be added as a Committer.

Command-Line

You will need to first install Subversion and Maven.

  • SVN checkout
    • Read only: svn checkout http://gdata-java-client.googlecode.com/svn/branches/2/gdata <some path>/gdata
    • Commit access: svn checkout https://gdata-java-client.googlecode.com/svn/branches/2/gdata <some path>/gdata
  • cd <some path>/gdata
  • Install: mvn install

Eclipse 3.5

You will need to first install Eclipse, the Subversion plugin, the Maven plugin, and the Android plugin.

  • Preferences: Android; setup SDK location
  • SVN checkout
    • File > Import...
    • Select "SVN > Project from SVN" and click "Next"
    • "Checkout Project from SVN repository" or "Enter Repository Location Information"
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn
        • Commit access: https://gdata-java-client.googlecode.com/svn
      • Click on "Next"
    • Select Resource
      • URL:
        • Read only: http://gdata-java-client.googlecode.com/svn/branches/2/gdata
        • Commit access: https://gdata-java-client.googlecode.com/svn/branches/2/gdata
      • Click on "Head revision" and click on "Finish"
    • Check Out As
      • Click on "Finish"
  • Install
    • Right-click on project gdata
    • Run As > Maven install

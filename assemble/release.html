Release Process

  • Update instructions.html
  • Assemble for Distribution: mvn clean source:jar javadoc:jar assembly:assembly
  • Upload to http://code.google.com/p/google-api-java-client/downloads/entry
    • Summary: Google API Java Client Library ${project.version}
    • File: google-api-java-client/target/google-api-client-${project.version}-java.zip
    • Labels: Type-Archive, OpSys-All, and Featured
    • click Submit file
  • Deprecate any obsolete libraries.
  • Create Mercurial Tag: hg tag ${project.version}
  • Put it up on Maven central repository?
  • Update javadoc project: copy target/site/apidocs to ${project.version} in javadoc project, and update javadoc/index.html to the new version
  • Update http://code.google.com/p/google-api-java-client/admin
